# Mavric Website

Proyecto frontend con Vite + React para ejecucion local.

## Requisitos

1. Node.js 18+ recomendado
2. npm

## Desarrollo local

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar entorno de desarrollo:

```bash
npm run dev
```

3. Abrir en el navegador la URL que imprime Vite en consola (normalmente `http://localhost:5173`).

## Scripts utiles

- `npm run dev`: servidor de desarrollo
- `npm run build`: build de produccion
- `npm run preview`: previsualiza el build
- `npm run lint`: analiza el codigo
